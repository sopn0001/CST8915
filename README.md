## Technical Explanations

### Order Service (Node.js)

This is a microservice written in Node.js with the purpose of managing customers orders, integrating and interacting with RabbitMQ for message queuing. There are few configuration steps to go through in order to run the application. Once application is running, we can test the endpoint using an rest client. 

### Product Service (Rust)

This is a microservice written in Rust.js with the purpose of handling product listings. There are few configuration steps to go through in order to run the application. Once application is running, we can test the endpoint using an rest client. 

### Store Front (Vue.js)

This section of the project is front-end web application where customers can browse and place order products. The application is written in Vue.js. There are few configuration steps to go through in order to run the application. Once application is running, we can test it a browser.
